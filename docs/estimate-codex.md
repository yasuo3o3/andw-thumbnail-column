# 開発見積もり書 — andw-thumbnail-column

## 前提
- 対象: WordPress 6.0+/PHP 8.0+ 向けプラグイン `andw-thumbnail-column`
- 根拠資料: `docs/SPEC.md` / `docs/PHASE-PLAN.md`
- 体制: フリーランス WordPress プラグイン開発者 1名
- 金額は税抜

## 1. 作業項目の分解（Phase + 設計・レビュー・テスト）

| 区分 | 作業項目 | 内容 |
|---|---|---|
| 設計 | 要件整理・実装設計 | 仕様確認、フック設計、設定データ設計、セキュリティ方針の確定 |
| Phase 1 | プラグイン骨格 | メインファイル、初期化、有効化時デフォルト設定 |
| Phase 2 | 設定画面 | Settings API、投稿タイプ別のON/OFF、サニタイズ |
| Phase 3 | サムネイル列追加 | 投稿タイプごとの動的フック、列描画、placeholder、管理画面CSS |
| Phase 4 | アンインストール処理 | `uninstall.php`、オプション削除 |
| テスト | 動作・品質確認 | 手動テスト、互換確認、`php -l`/`phpcs`/`phpstan` 実施 |
| レビュー | 仕上げレビュー | コードレビュー、修正反映、最終確認 |

## 2. 各項目の工数（人日）

| 区分 | 人日 |
|---|---:|
| 設計 | 0.6 |
| Phase 1 | 0.3 |
| Phase 2 | 0.5 |
| Phase 3 | 0.5 |
| Phase 4 | 0.1 |
| テスト | 0.8 |

| **小計** | **3.2** |

## 3. 想定単価と合計金額（JPY）

- 想定単価: **80,000円 / 人日**（実務経験のあるフリーランス WordPress 開発者の中位レンジ想定）

| 項目 | 計算 | 金額 |
|---|---:|---:|
| 小計 | 3.2人日 × 80,000円 | 256,000円 |

## 4. リスクバッファ

- バッファ率: **25%**
- 根拠:
  - サイトごとのカスタム投稿タイプ構成差異
  - 他プラグインの一覧列カスタマイズとの競合調整
  - テスト環境差分（テーマ・権限・既存データ）による追加検証

| 項目 | 計算 | 金額 |
|---|---:|---:|
| リスクバッファ | 256,000円 × 25% | 64,000円 |
| **見積合計** | 256,000円 + 64,000円 | **320,000円** |

## 5. 納期の目安

- 実作業日数: **約4.0人日（3.2人日 + バッファ相当）**
- カレンダー換算: **着手から約1.5〜2週間**
  - 例: 2026年3月16日（月）着手の場合、**2026年3月25日（水）〜3月30日（月）** 納品目安
